Kendi sunucunuzu kullanmayı tercih ediyorsanız, 80 ve 443 portlarının doğru şekilde yönlendirildiğinden emin olun. Böylece genel ip adresi yönlendiriciye değil, sunucuya yönlendirilir. İşiniz bittiğinde kaydetmek için ctrl s, nano'dan çıkmak için ctrl x e basın

```bash filename expandable
nano /etc/tor/torrc
```

içerisine aşşağıdaki yazıp aynı şekilde kaydedip çıkın

```bash
HiddenServiceDir /var/lib/tor/hidden_service/
echo 'HiddenServicePort 6667 127.0.0.1:6667
```

Ardından

```bash
systemctl restart tor
cat /var/lib/tor/hidden_service/hostname
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx.onion
```

Artık bu adresi kullanarak ırc sunucunuza bağlanabilirsiniz, şimdi inspircd'yi yeniden başlatın

```bash
systemctl status inspircd 
systemctl restart inspircd

## client:

sudo apt install irssi hexchat tor

sudo systemctl start tor
```
